Const forEach

  • forEach(collection: any, iterator: (value: any, key: any) => void, allowNullValues?: boolean): void
    • Defined in functions/src/iterators/forEach.ts:43

    Iterate over just about any iterable object type - primarily for node lists, arrays and objects, but will also yield results for single elements such as elements (nodes), strings or numbers.

    example

    Iterating through an array:

    import { forEach } from "@baggie/functions";

forEach([value1, value2, value3], (value, index) => {
    // Loop through the values in the array
});
    example

    Iterating through an object:

    import { forEach } from "@baggie/functions";

forEach({"key1": value1, "key2": value2}, (value, key) => {
    // Loop through the values in the object
});
    example

    Iterating through a DOM-collection:

    import { forEach } from "@baggie/functions";

const collection = document.querySelectorAll("div");
forEach(collection, element => {
    // "element" is a reference to the current DOM-element.
});
