Coordinates - Angle

Const getAngle

  • getAngle(from: CoordinatesXY, to: CoordinatesXY, options?: undefined | { clockwise?: undefined | false | true; offset?: undefined | number }): number
    • Defined in math/src/coordinates/angle/getAngle.ts:32

    Get the angle from one set of coordinates to another.

    remarks
    • Be aware that angles in coordinate systems move counter clockwise, hence this is the default setting. But you can change that in the options argument by setting clockwise: true.
    example

    Basic usage:

    import { getAngle } from "@baggie/math";

const from = { x: 10, y: 10 };
const to = { x: 20, y: 20 };

const angle = getAngle(from, to);
// angle = 135

const angleWithOffset = getAngle(from, to, { offset: 400 });
// angle = 175

const angleClockwise = getAngle(from, to, { clockwise: true });
// angle = 225

    Parameters

    • from: CoordinatesXY
    • to: CoordinatesXY
    • Optional options: undefined | { clockwise?: undefined | false | true; offset?: undefined | number }

    Returns number
